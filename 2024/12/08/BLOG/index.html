1.3 强调

你可以使用星号 * 或下划线 _ 来实现文本的斜体和加粗。

  • 斜体：用单个星号或下划线包围文本。

  • 加粗：用两个星号或下划线包围文本。

  • 加粗斜体：用三个星号或下划线包围文本。

1.4 列表

Markdown 支持无序列表和有序列表。

  • 无序列表：使用星号 *、加号 + 或减号 -。

  • 有序列表：使用数字加点。

1.8 代码

Markdown 支持插入代码块和行内代码。

2.2 Front Matter（文件头信息）

每个 Hexo Markdown 文件的开头都会有一段 front matter，这是一个 YAML 格式的元数据，用来配置文章的标题、日期、标签、分类等信息。这个部分用三个连字符 --- 包裹，位于文件的最顶部。

常见的 Front Matter 配置项包括：

  • title：文章标题。
  • date：发布时间。
  • tags：文章标签。
  • categories：文章分类。
  • excerpt：文章摘要。
  • comments：是否开启评论功能。
